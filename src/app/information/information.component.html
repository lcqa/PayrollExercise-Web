Setup

Pre requisites
  • Node JS Version: 16.16.0
  • Anguler Version: 16.2.0
  • NPM Version: 8.5.4
  • VS Code or any code editor
  • VS 2022 for Backend
  • .NET 6 SDK

Configuration

Environment
Change the value of baseApiUrl that can be found in the app/environment folder of the web project, the default value is http://localhost:5185/api.
The port configuration of the backend can be found in the API project, path: Payroll.API/Properties/launchSettings.json
This configuration allows the Web Application to commyunicate with the Backend Project.
Running the App
Type the angular command ng serve -c -o
For the backend, in Visual Studio Press F5 to start debugging
